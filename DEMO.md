# 🎬 SpiderSense Demo Guide

## 📸 Screenshots & Demo Videos

### For Social Media Posts

**Recommended Content to Capture:**

1. **Threat Detection Demo** (30-60 seconds)
   - Show yourself on camera
   - Slowly move your hand toward the camera
   - Show the threat ring pulsing and changing colors
   - Throw a punch - watch it trigger "THREAT DETECTED"

2. **Gesture Control Gameplay** (30-60 seconds)
   - Split screen showing your hands + game
   - Demonstrate each gesture:
     - Left hand → Spider-Man moves left
     - Right hand → Spider-Man moves right
     - Both hands up → Jump
     - Fist → Shoot web
     - Web-swinging gesture

3. **Boss Battle** (15-30 seconds)
   - Show boss health bar
   - Defeat a boss
   - Level up animation

## 🎥 Recording Tips

### For Reddit/Instagram
- **Resolution**: 1080p minimum
- **Format**: MP4 or GIF
- **Length**: 30-60 seconds max
- **Show**: Split screen (camera + game) or picture-in-picture

### For Twitter/X
- **Length**: 15-30 seconds
- **Format**: MP4
- **Captions**: Add text overlays explaining gestures

### Screen Recording Tools
- **macOS**: QuickTime Player (Cmd+Shift+5)
- **Windows**: Xbox Game Bar (Win+G)
- **Linux**: OBS Studio
- **Browser**: Chrome DevTools (Cmd+Shift+P → "Screenshot")
